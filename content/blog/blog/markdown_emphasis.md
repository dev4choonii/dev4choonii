## 강조(Emphasis)

```
*Emphasis*
_Emphasis_
**Emphasis**
__Emphasis__
~~Emphasis~~

__*Emphasis*__
_*Emphasis*_
~~*Emphasis*~~
~~**Emphasis**~~
```

*Emphasis*  
_Emphasis_  
**Emphasis**  
__Emphasis__  
~~Emphasis~~

_*Emphasis*_

```

```

> 문장 사이는 `*`로 처리.  
> 문장 사이는 띄워쓰기.

You'll **Never** Walk Alone.
